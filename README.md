# ECO-PROJECT
Телеграм бот с игрой, где пользователь должен найти причины глобального потепления по подсказкам в тексте, играя в детектива. В боте будут кнопки, с помощью которых можно будет прийти к какому-то определенному результату.

## Тип моего проекта:
> текстовый квет Телеграм-бот.
> Жанр: детектив на основе реальных кейсов ООН
> Целевая аудитория: люди младше 40 лет (школьники, студенты)

## Библиотеки, которые я буду использовать
- python- telegram
- random

## Референсы, которые мне пригодятся
- Документация python- telegram
- Пример текстового квеста
- Текста ООН

## Гайды-статьи, где есть полезная для меня информация
> Доклады ООН на тему климат и глобальное потепление
